== DESCRIPTION ==

This is a child theme of the WordPress Twenty Nineteen theme. This allows us to add more features to the theme as well as adjust the look and feel of it. It is called "Flat" becuase the theme defines a nice rich color palette for building sites with colored sections and doesn't use gradients which equate to what is referred to as a "flat style" look.

For more information on this theme, go to [http://xtremelysocial.com/wordpress/flat/].

== INSTALLATION ==

1. Download and install the parent theme, Twenty Nineteen
2. Download this theme into its own directory in /wp-content/themes/
3. Activate the theme through the Appearance->Theme menu in WordPress admin
4. Read the notes below about how to use this theme
